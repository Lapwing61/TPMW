===================================
ТПМ - погодный модуль 
===================================
Назначение: Формирование XML файла по данным погодного сервиса Dark Sky для городов из списка

Использование: Модуль берет входные данные из трех источников
	1. Командная строка. Параметры
	   -h или --help				Выдается краткая справка по параметрам командной строки и завершение работы модуля
	   -cXXX или --config XXX		Задается имя кофигурационного файла (по умолчанию config.ini) 
	   -lXXX или --list XXX			Задается имя файла городов (по умолчанию city.csv)
	   отсутствие параметров эквивалентно (-c -l)
	   при отсутствии параметра -h начинается выполнение модуля
	2. Кофигурационный файл (по умолчанию config.ini)
	    Секция [Provider]
		name						Имя домена (по умолчанию api.darksky.net)
		id							-- не используется --
		key							Секретный ключ (полученый при регистрации на погодном сервере)

		Секция [Output]
		type						-- не используется --
		file						имя XML файла
		dir							имя директории с выходным XML файлом
			или
		site						имя сайта для размещения выходного файла
		login						имя пользователя (логин) на сайте
		password					пароль на сайте
		при наличии непустого поля dir поля site, login, password игнорируются
		при отсутствии и dir и site - вывод производится на экран.

		Секция [Icons_src]			Пути до файлов с погодными иконками 

		Секция [Icons_crc]		    Контрольные суммы
	3. Список городов (по умолчанию city.csv)
	   Представляет собой csv файл с разделителем ";" (точка с запятой)
       Поля по очереди:
	    Имя города
		ID города
		широта
		долгота
		zoom1
		zoom2
		Файл не должен содержать пустых строк 

Результат: Выходной файл формируется согласно полям dir и file (или site и file) конфигурационного файла 
и представляе собой xml файл заданного формата
